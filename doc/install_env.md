### docker

安装Docker CE：

`sudo yum install -y docker-ce  # CentOS/Red Hat`
`# 或者`
`sudo apt install -y docker-ce  # Ubuntu/Debian`

启动Docker服务：
`sudo systemctl start docker`

设置Docker开机自启：
`sudo systemctl enable docker`

### mysql数据库

`docker pull mysql`
